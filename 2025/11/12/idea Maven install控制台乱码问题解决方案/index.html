idea Maven install控制台乱码问题解决方案

发表于2025-11-12|更新于2025-12-24|牛马工具箱开发问题
|总字数:148|阅读时长:1分钟|浏览量:

失败尝试：设置idea编码

打开 idea, 点击 File/文件 -> Settings/设置 找到 编辑器 - 文件编码，全部设置为 UTF-8
这个是网上到处都是的常规设置，在我的环境里是没用的

成果解决：Maven环境变量配置中文

打开 idea, 点击 File/文件 -> Settings/设置 -> Build,Execution,Deployment/构建、执行、部署 -> Build Tools/构建工具 -> Maven -> Runner/运行程序
(也可以在操作系统的环境变量里设置)
Name=MAVEN_OPTS
Value=-Dsun.stdout.encoding=UTF-8 -Dstderr.encoding=UTF-8 -Dconsole.encoding=UTF-8 -Dfile.encoding=UTF-8
